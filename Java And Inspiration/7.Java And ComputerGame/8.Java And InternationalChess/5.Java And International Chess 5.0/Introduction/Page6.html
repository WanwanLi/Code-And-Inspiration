Exchange Strategy
Based On Min Max Method,the computer consider how to go to lose less.But I think you should know how much you can gain.Thus,your evaluation should be:nextScore = your Score - your Opponent's MaxScore.And maximize your nextScore.I call this Exchange Strategy!I think,this Strategy will ensure you to gain more instead of gain nothing and lose nothing.Sometimes it make the computer more intelligent .So 6.0 version is based on this Strategy.
